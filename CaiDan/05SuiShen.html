                                使用搜索查阅

                                系统类的东西过于昂长，建议配合图片，多使用Ctrl+F查阅需要的内容。
                                你可以输入菜单的英文直接查找。

                                当你按下Esc时，你将在屏幕中心看到如下图的界面。（VR端的随身菜单会出现在你按下菜单那键的手中）

                                # 主界面

                                一层菜单的按钮翻译如下：

                                点击表格中的名称查看相关教程。

                                按钮说明
                                广告/公告区官方用来推送资讯的栏位
                                系统菜单按钮位于随身菜单右上角，点击后会根据当前随身菜单页面打开系统菜单。

                                化身列表点击后打开系统菜单的化身页面
                                联系人点击后打开系统菜单的联系人页面，
                                联系人可以理解成你的好友列表。但是你还可以看到你的黑名单列表。
                                群组列表点击后打开群组列表，
                                群组可以理解成社团，公会。
                                随便加入，但建设群组是氪金功能哦！
                                回家点击后，经过二次确认回到你的初始地图。
                                默认情况下，初始地图是VRChat Home
                                重生点击后，立即回到所处地图的出生点
                                可被世界作者更改
                                选择用户点击后菜单隐藏，屏幕中心可以选中用户
                                鸡肋功能，凑数用的。
                                在菜单状态下，鼠标移出菜单范围就可以选择其他用户
                                安全模式点击后可快速切换安全模式
                                麦克风开关点击后切换麦克风的开关
                                首页切换到随身菜单的首页
                                通知切换到通知页
                                好友申请，申请加入等消息会在此处
                                当前房间信息切换到房间信息页
                                相机菜单切换到相机菜单
                                音频设置切换到音频设置菜单，可以设置各种音量大小和音频设备等功能
                                设置切换到快捷设置页
                                紧急安全模式点击后进入紧急安全模式，屏蔽所有非好友用户的化身
                                快捷键为 Shfit+ESC
                                快捷键只能启动，不能关闭

                                # 通知界面

                                在通知界面，系统将为您统一推送各种信息：

                                • Invites申请界面
                                  用于接收您收到的邀请信息，当有人邀请你加入他的房间，或申请加入你的房间，这里会显示通知
                                • Friend Requests界面
                                  用于接收您收到的好友信息，如果有人申请与您成为好友，这里会显示通知
                                • Group界面
                                  用于接收您收到的群组信息，如果有人申请加入您的群组，或你的群组公开发布信息，这里会显示信息
                                • Other界面
                                  系统信息界面，如果有踢人等投票活动会在这里显示信息

                                # 位置信息

                                位置界面会显示您当前所处房间的信息。

                                • 世界封面图
                                  点击世界封面图，将会打开系统菜单，详细介绍世界信息，与右上角按钮功能一致。
                                • 重新加入
                                  你将重新加入这个房间，主要用于解决某些可能的Bug （重启解决99%的问题）
                                • 新建房间
                                  你将以这个世界为目标，新建一个房间，该操作详见：
                                • 收藏世界
                                  点击后，将会选择一个世界收藏夹将世界放入其中。
                                • 房间用户列表
                                  你可在此查看所有位于该房间中的用户信息，包括你黑名单中屏蔽的用户。

                                # 音频控制

                                该界面可以控制VRChat中所有的音频属性。

                                属性说明
                                所有音量大小控制整个VRChat的音量大小
                                菜单音量大小单独控制ESC下所有菜单的音量，例如按钮反馈音
                                世界音量大小单独控制场景的音量，例如房间中的背景音乐、视频播放器的音量等
                                用户语音音量大小单独控制其他玩家从麦克风发出的音量的大小，该属性可以针对每个用户单独调控
                                化身音量大小单独控制由用户模型发出的音量，例如模型特效的声音
                                重置他人音量大小点击该按钮时，会将你设置的所有他人语音音量大小重置为100%（详见“他人菜单”）
                                切换麦克风点击按钮显示您所有的麦克风设备，选择一个作为VRC的麦克风
                                输出音量大小控制您说话的音量大小
                                降噪开启或关闭VRC内置的降噪系统
                                输出阈值如果您麦克风收到的的声音小于阈值，VRC将不予发声，适用于环境音过大时，在一定音量下不发声

                                # 麦克风模式

                                VRC的麦克风支持多种操作模式

                                模式说明
                                Toggle切换模式，通过按下V键切换麦克风的开关。同时需要设置每次进房间的麦克风属性
                                - Default On ： 每次进入房间时，把麦克风打开
                                - Default Off ： 每次进入房间时，把麦克风关闭
                                - Keep Last State ： 不自动切换，保留上一次操作的模式
                                Always ON永远保持开启麦克风，且无法关闭
                                Always OFF永远保持关闭麦克风，且无法开启
                                Push to Talk按键发言，只有按下V时才打开麦克风，松开V就关闭

                                # 设置面板

                                VR独立设置

                                如果您使用VR设备，该界面将会多出额外的设置，额外的部分请见《VR模式随身设置菜单》

                                属性说明
                                关闭游戏二次确认后，关闭VRChat应用程序
                                允许他人克隆化身当开启此功能，且您的化身是公开化身，则他人可以克隆您的化身使用（变成你的样子）
                                自动关闭克隆当您切换房间时，自动将克隆功能关闭，防止忘记关
                                启用手势动画当启动时，允许您通过Shift+F1-F8 / VR手柄 / 其他设备来操作手势于表情，反之禁止。
                                停止化身的互动效果关闭您化身的所有互动效果，任何人不可互动
                                谁可以互动当您开启了互动效果，则：
                                - Everyone:任何人都可以互动你的化身
                                - Friends:只有你的好友可以互动你的化身
                                - Nobody: 没有人可以互动你的化身
                                自我互动功能是否允许你自己互动自己的化身
                                化身显示范围当开启时，以您的位置为圆心，只显示N米范围内的其他用户化身
                                最大化身显示数量当开启时，您最多同时可见N个其他用户化身
                                永远显示好友化身只要是你的好友，永远显示他们的化身，该功能不占用上述两个规则
                                安全设置优先于该设置如果您单独设置了某人的化身可见，则以该设置为准，该功能不占用上述两个规则
                                清除所有聊天气泡按键按钮瞬间删除所有人的聊天气泡（仅为本地）
                                显示聊天键盘打开聊天键盘，等同于按下Y键
                                气派可见度你能看到谁的气泡：
                                - Everyone:可以看到所有人的
                                - Friends：只能看到你好友的气泡
                                - Mine：只能看到自己的气泡
                                房间亮度设置房间的渲染亮度，方便睡觉
                                泛光亮度抑制地图的后处理泛光，让眼睛好受点
                                防3D眩晕隧道建议用于VR 当您移动时，压制您的视野，这将极大缓解3D眩晕，有3种模式可选：
                                - None：不启动该功能
                                - Low 低强度隧道
                                - High 高强度隧道，且视野边缘会有暗角
                                自由放置菜单VR专用 允许您在空间中自由的放置菜单
                                日志系统是否将您的运行记录打包成日志，方便出现问题是官方检查
                                持续显示FPS和延迟启动后，将在你的菜单左上方持续显示FPS和延迟
                                注意，菜单本身会降低10FPS左右
                                汇报Bug点击打开汇报Bug界面

                                # 聊天气泡高级设置

                                该部分允许您设置更多与聊天气泡有关的属性。

                                属性说明
                                ChatBox Position气泡显示的位置，有多种模式可选：
                                - Above：显示在上面（头顶）
                                - Forward：显示在前面
                                   - Chatbox Height： 设置气泡相对化身有多高
                                ChatBox Size设置气泡的大小
                                ChatBox Opacity设置气泡的透明度
                                Display Duration气泡持续多久
                                Notification Volume设置气泡的提示音大小，可直接关闭
                                Show Own ChatBox是否可见自己的气泡

                                # 颜色高级设置

                                该部分允许你对菜单的颜色进行部分设置，且可将设置套用到游戏中。

                                属性说明
                                Filter Intensity调整强度
                                Apply to World将颜色调整使用到整个游戏中
                                Color Filter调整模式：
                                - No Filter：不调整
                                - Protanopia：红色色盲模式
                                   - Simulate Color Blindness：模拟色盲功能
                                - Deuteranopia：绿色色盲模式
                                   - Simulate Color Blindness：模拟色盲功能
                                - Tritanopia：蓝色色盲模式
                                   - Simulate Color Blindness：模拟色盲功能
                                - Grayscale：灰度（黑白）调整
                                - Retro LUT：复古色调 （图都发绿了！）

                                # 铭牌高级设置

                                该部分用于控制用户铭牌的显示效果。

                                铭牌是什么？

                                是每个人头上的这个：

                                属性说明
                                Nameplate Visibility铭牌显示模式
                                - Standard：显示详细的信息，包含用户名字和签名，如用户是你的好友，名字是黄色的
                                - Icon：仅显示用户头像和名称缩写，如果用户是你的好友，名字是黄色的，且进行特殊标记
                                -Hidden 隐藏铭牌，不显示任何内容
                                Show Group Banner是否在铭牌上显示用户群组信息
                                Quick Menu Info当你按下ESC打开菜单时，是否强制显示最详细的信息
                                HUD Detall Level控制你HUD的显示内容：
                                - Verbose：最详细的
                                - Minimal:简单内容显示
                                - Off： 不显示HUD，等同于按下Ctrl+H
