                                撰写日

                                VRC每次更新都可能会添加新的系统，该页面撰写日为：2023/8/20 9:00

                                在任意随身菜单页面，点击右上角的放大按钮，打开系统菜单。
                                推荐在首页打开！

                                # 系统菜单


                                系统菜单是一套用于显示所有详细内容的菜单，与随身菜单不同，它更大，且在VR模式下不随用户移动。

                                • 随身菜单的内容就是由系统菜单精简而来。
                                • 随身菜单部分操作是快捷跳转系统菜单。

                                # 系统菜单分页


                                从左到右分别是：

                                • 首页
                                • 信息面板
                                • 个人面板
                                • 世界面板
                                • 化身面板
                                • 联系人面板
                                • 群组面板
                                • 安全设置面板
                                • 系统设置面板
                                • 搜索系统
                                • 会员系统
                                • 官方手册面板

                                # 首页

                                系统菜单的首页主要会显示一些整理信息，从上往下（从左往右）依次为：

                                • 当前官方的推送消息
                                • Friends Online：你有多少好友在线
                                • Invites：你收到了多少房间邀请
                                • Friend Requests：你收到了多少好友邀请
                                • Recentiy Visited Online Friends：你当前在线的，且可以直接加入房间的好友列表
                                • Recent Worlds：你去过的世界足迹
                                • Recently Updated Favorite Worlds：你收藏的世界中，最近更新的世界
                                • Recently Updated Favorite Avatars： 你收藏的化身中，最近更新的化身
                                • Recently Played With：最近和你在一个房间的用户

                                # 信息面板

                                信息面板主要是为您查阅收到的各种信息。
                                左侧为信息的分类与功能：

                                • All：所有信息
                                • Friend Requests：您收到的加好友的申请信息
                                • Invites：您收到的邀请您去其他房间的申请信息
                                • Invite Requests：你收到的申请加入您的房间的申请信息
                                • Replies：他人给你的答复信息
                                • Group：你的群组信息
                                • Other：其他信息，例如投票踢人，官方消息等
                                • Edit Messager：编辑您的快速答复

                                右上角的选项：

                                • Hide Images：隐藏图片，您将会看不到所有信息中的图片
                                • Decline All：删除所有信息

                                # 个人面板


                                该面板允许您调整自己的各种个人信息。
                                分别为：

                                名称描述
                                头像封面你当前正在用的化身封面图
                                Status你当前的社交状态，点击后可以更改你的社交状态与社交签名
                                Trust Rank你当前的社交等级，如果你超过了橙色，你可以点击伪装成绿色
                                Bio你的个人介绍，点击后会打开一个输入框。
                                由于VRC的PC端适配依然有问题，建议在别处打好字直接复制进来
                                Location你当前所在的房间信息，包括世界封面/世界名称/房间号/房间类型/房间人数/满额容量。
                                点击后打开世界详情页面
                                Languages您的社交语言，点击+号来添加您的沟通语言,再次点击可以删除语言
                                Links你的社交网址，你可以输入你的B站个人空间、推特链接、ins链接等
                                别人点击直接打开浏览器，自己点击删除链接
                                Badges你的成就与徽章，如果你有的话会显示在这里
                                Groups你的展示社团，点击View All可以查看你加入的所有社团。
                                Avatars你可以在此看到你自己的所有化身
                                Uploaded World你可以在此看到你上传的所有世界
                                世界收藏夹根据您对世界分类收藏夹的设置，此处会对其他人公开您的世界收藏

                                # 世界面板

                                世界面板是用来展示当前VRChat中的世界的面板。同时他也可以操作你的收藏夹。

                                个人视图默认被收起

                                在左侧的分栏中，My Worlds是您的个人世界视图，默认被收起。

                                • 在整个左侧分栏的最上方，显示为您当前所在的位置，即当前所在的房间名称。
                                  点击后打开世界详情页面
                                • 左侧分栏的第二个按钮，为打开搜索界面。输入字符以搜索目标内容。
                                • 在界面的右上角可以切换显示模式

                                个人视图默认被收起

                                点击任何一个世界卡片，跳转到世界详情页面

                                # 个人视图

                                当My Worlds展开后，你将看到自己的世界视图。从上往下为：

                                名称描述
                                Uploaded你上传的世界列表
                                Recently Visited最近访问列表，按照你曾经去过的世界依次显示，最多记录18条
                                最下方会显示您的初始地图
                                (Recently) Updated Favorites您收藏夹中最近有更新的世界，你可以经常看看更新了啥
                                收藏夹1-4您有4个世界收藏夹，每个收藏夹限制100个世界
                                在每个收藏夹的右上角，您可以设置收藏夹名字，清空文件夹，以及公开性：
                                - Private ：私有的，谁也无法看到
                                - Friends ：只有你的好友可以看到
                                - Public : 完全公开

                                # 系统视图

                                默认的World视图为系统推荐的各种分类列表，从上往下为：

                                名称描述
                                VRCat's Variety BoxVRChat官方的一个随机推送列表
                                Trending热门地图列表
                                Avatar Worlds - Trending热门的化身地图，在该列表的世界里可以找到各种公开化身
                                Game - Hot热门的游戏地图
                                Spotlight - Cross Platform跨平台世界列表，该列表显示的世界均支持一体机用户进入
                                Spotlight - PCPC专属世界，该列表显示的世界一般质量较高，一体机无法进入
                                New新世界
                                Avatar Worlds - New新的化身世界
                                Updated Recently新上传的世界
                                Active活跃的世界，一般来说，这个世界的房间的人数持续性活跃
                                Community Labs处于实验室的地图
                                Random随机推荐列表
                                Avatar Worlds - Random随机推荐给你一些化身世界
